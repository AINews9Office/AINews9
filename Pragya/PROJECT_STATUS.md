# Phase 1 MVP Scope

The MVP will deliver:

- Guided Learning Experience
- AI Journey Selection
- Persona Selection
- Topic Suggestions (Static MVP)
- Session Context
- Gateway
- Basic Understanding Layer
- Basic Knowledge Retrieval
- Guided AI Conversations
- Continue Learning Recommendations

Out of Scope:

- Authentication
- Learning History
- Adaptive Personalization
- Quizzes
- Certifications
- Gamification
- Voice
- Mobile Applications

These capabilities belong to future phases.
