# AINews9 Knowledge Base (PKB)

## Purpose

The AINews9 Knowledge Base (PKB) is the single source of truth for the AINews9 platform.

It contains the product vision, strategy, architecture, execution plans, standards, and decision history required to build and evolve AINews9.

Every AI agent, developer, and contributor should start here before making any changes.
