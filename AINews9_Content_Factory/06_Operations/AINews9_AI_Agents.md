# Information Density Rules

AINews9 cornerstone images should communicate meaningful knowledge.

Images should not rely solely on decorative visuals.

Minimum Educational Elements:

✓ Title

✓ Subtitle

✓ Visual Illustration

✓ Concept Explanation

✓ Example

✓ Benefits or Features

✓ Learning Outcome

Preferred Educational Elements:

✓ Timeline

✓ Workflow

✓ Comparison

✓ Checklist

✓ Diagram

✓ FAQ Snippets

## Educational Section Count Rule

Cornerstone Educational Posters should contain:

Minimum:

* 1 Main Visual
* 4 Educational Panels

Preferred:

* 1 Main Visual
* 6–10 Educational Panels

* Definition Panel
* Features Panel

* Use Cases Panel
* Workflow Panel

Reject layouts containing:

✗ Only title and illustration

✗ Hero-banner style layouts

✗ Decorative marketing visuals

✗ Fewer than 3 educational sections

Rule:

A cornerstone image should resemble a mini-learning poster or infographic rather than a blog thumbnail.

# Hindi Technology Terminology Policy

Do not perform literal translation of established technology terms.

Preferred:

* एआई
* चैटजीपीटी
* मशीन लर्निंग
* जनरेटिव एआई
* लार्ज लैंग्वेज मॉडल
* क्लाउड कंप्यूटिंग
* प्रॉम्प्ट इंजीनियरिंग
* डेटा प्राइवेसी
* साइबर सुरक्षा
* ऑटोमेशन
* डिजिटल ट्रांसफॉर्मेशन

Avoid:

* कृत्रिम बुद्धिमत्ता
* यंत्र अधिगम
* बादल संगणना
* संकेत अभियांत्रिकी

Unless educational context specifically requires mentioning these terms.

# Writing Style Rules

## Rule 1

Use short sentences.

---

## Rule 2

Use conversational language.

---

## Rule 3

Explain complex ideas simply.

---

## Rule 4

Use practical examples.

---

## Rule 5

Avoid unnecessary English when a commonly understood Hindi expression exists.

---

## Rule 6

Avoid excessive formal language.

Avoid:

* प्रयुक्त
* उपर्युक्त
* उक्त
* अधोलिखित
* तदनुसार

Prefer simple everyday language.

---

## Rule 7

Write the way modern Indians speak and consume technology.

Good:

"आज एआई शिक्षा, स्वास्थ्य और बिजनेस में तेजी से उपयोग हो रहा है।"

Bad:

"आज कृत्रिम बुद्धिमत्ता शिक्षा, स्वास्थ्य एवं व्यापार में तीव्र गति से प्रयुक्त हो रही है।"

---

## Rule 8

Content should be easy to read aloud.

If a sentence sounds unnatural when spoken, simplify it.
